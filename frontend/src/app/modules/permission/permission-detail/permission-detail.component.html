help Héritage des permissions

GeoNature possède un mécanisme d'héritage des permissions via les groupes d'utilisateurs et/ou les modules dont voici les principales caractèristiques :

  • Les membres d'un groupe héritent des permissions de son ou ses groupes d'appartenance.
  • Le module GEONATURE est le module principal par défaut dont tous les autres héritent.
  • Si un module est associé à des objets spécifiques ceux-ci héritent des permissions de leur module.
